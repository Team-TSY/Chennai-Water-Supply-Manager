# DATASETS FOLDER

This folder contains the datasets of 4 reservoirs :
1. Poondi
2. Cholavaram
3. Chembarambaakam
4. Redhills

The dataset contains the following fields:
* Storage (in MCFT)
* Inflow (in cusecs)
* Rainfall (in mm)
* Outflow (in cusecs)
* Date
* nth day in year


**Dataset credits**: Chennai Metropolitian Water Supply & Sewage Board : https://chennaimetrowater.tn.gov.in/
